# Java Notes Manager

## 📌 Objective
A simple text-based notes application using Java File I/O.

## 💻 Features
- Add new notes
- View saved notes
- Clear all notes

## 🧰 Tools Used
- Java
- VS Code
- Terminal

## 🔍 Key Concepts
- FileReader, FileWriter
- BufferedReader
- Exception handling

## 📁 How to Run
```bash
javac NotesApp.java
java NotesApp
